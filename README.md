# Projeto 3

Este é o repositório do projeto 3.

## Configuração do Git

Este repositório foi configurado para sempre atualizar os arquivos do projeto. As seguintes configurações foram implementadas:

1. Arquivo `.gitignore` adicionado para excluir arquivos desnecessários
2. Hook `pre-commit` configurado para adicionar automaticamente todos os arquivos do projeto antes de cada commit

## Arquivos do Projeto

- `index.html`: Página principal
- `style.css`: Folha de estilo
- `README.md`: Documentação
- `image/`: Diretório de imagens
